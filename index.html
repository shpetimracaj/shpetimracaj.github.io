Computer Science Capstone Enhancement Narrative

The artifact I chose for this milestone is the pyramid problem implemented in C++. Initially created as part of an earlier assignment, this code simulates a pen using 3D graphics. It provides a basic structure to visualize geometric shapes and their properties.

I included this artifact in my ePortfolio because it showcases my skills in C++ programming and graphics rendering. The original implementation demonstrated fundamental concepts of 3D modeling; however, I recognized opportunities for enhancement. In this enhancement, I added more lighting textures, allowing for a more realistic visual representation of the pyramid. I also enabled user interaction with the pen, enhancing the overall experience. Additionally, I integrated a table to display relevant data about the pyramid's dimensions and properties, making the output more informative. The improved comments throughout the code facilitate better understanding and maintainability.

Enhancing this artifact was a rewarding learning experience. Through the process, I gained a deeper understanding of lighting techniques in 3D graphics and how they impact visual realism. Additionally, adding user interaction helped me explore event handling and responsiveness within graphical applications. One of the challenges I faced was ensuring the new features worked seamlessly without compromising the existing functionality. I overcame this by thoroughly testing each enhancement incrementally, ensuring that any changes made were integrated smoothly. This experience has further solidified my ability to adapt and improve existing codebases while enhancing user experience.
